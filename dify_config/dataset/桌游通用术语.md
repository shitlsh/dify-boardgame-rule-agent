# 桌游通用术语与概念知识库

本文档面向**全品类桌游**（德式、美式、聚会、卡牌、战棋等），供规则提取与问答时统一术语、理解机制类别。  
说明：具体某款游戏的专有名词应以该款规则书为准；此处为**跨游戏通用**概念。

---

## 1. 术语使用与翻译原则

- **首次出现**：重要术语可写「中文（英文）」，如：工人放置（Worker Placement）。
- **已约定俗成的译名**：如「米宝」对应 Meeple，不必再强行改写。
- **机制名 vs 普通词**：如 Engine（引擎）在桌游语境常指「资源-行动的转化链条」，与「汽车引擎」无关。
- **VP / AP 等缩写**：在中文规则中可保留英文缩写并括号注明全称。

## 2. 游戏类型与体验（高频英文）

| 英文 | 中文 | 简要说明 |
|------|------|----------|
| Eurogame / German-style | 德式游戏 | 策略、计分、互动偏克制，常称「德式」 |
| Ameritrash / Ameritrash | 美式游戏 | 叙事、模型、冲突强，规则量可能较大 |
| Party Game | 聚会游戏 | 人数多、规则短、社交与娱乐性强 |
| Abstract | 抽象游戏 | 主题弱，规则极简，如棋类 |
| Wargame | 战棋 / 兵棋 | 模拟战斗、地形、单位 |
| Living Card Game (LCG) | 成长式卡牌游戏 | 固定扩展包、非随机卡包（相对 TCG） |
| Trading Card Game (TCG) | 集换式卡牌 | 随机卡包、二级市场 |
| Cooperative Game | 合作游戏 | 玩家共同对抗游戏系统 |
| Semi-cooperative | 半合作 | 既有合作又有个人胜负或阵营 |
| Solo / Solitaire | 单人模式 | 单人可玩 |
| 1v1 / FFA / Team | 单挑 / 混战 / 组队 | 对抗结构 |

## 3. 回合与流程结构

| 英文 | 中文 | 释义 |
|------|------|------|
| Round | 轮次 | 所有玩家各进行一次「回合」或等价流程后，称为一轮（部分游戏定义不同） |
| Turn | 回合 | 某一玩家连续行动的时间单位（定义因游戏而异） |
| Phase | 阶段 | 一轮内按顺序划分的步骤（如抽牌阶段、行动阶段） |
| Step | 步骤 | 比阶段更细的单位 |
| Initiative / Turn Order | 先手 / 行动顺序 | 谁先行动、顺时针或顺位编号 |
| Simultaneous | 同时选择 | 玩家同时暗置或同时展示，减少等待 |
| Upkeep | 维持 / 回合开始结算 | 自动触发的资源、标记刷新 |
| Cleanup | 回合结束清理 | 弃牌、重置标记等 |
| Pass | 跳过 / 宣告不再行动 | 本轮或本局不再执行某些行动 |
| Final Round / Last Round | 终局轮 | 触发游戏结束前的最后一轮或若干轮 |

## 4. 配件与版图

| 英文 | 中文 | 释义 |
|------|------|------|
| Board | 版图 / 主棋盘 | 公共游戏区域 |
| Tile | 板块 | 可拼接或随机摆放的地图片 |
| Tile Placement | 板块拼放 | 机制类别：放置板块改变版图 |
| Card | 卡牌 | 纸牌组件 |
| Deck | 牌库 / 牌堆 | 抽牌来源；**Draw pile** 常指待抽牌堆 |
| Discard Pile | 弃牌堆 | 使用后进入的牌堆，有时需洗牌回到牌库 |
| Hand | 手牌 | 玩家持有的牌 |
| Token | 指示物 | 小圆片、方块等代表资源或计数 |
| Meeple | 米宝 | 人形木质指示物，常代表工人或玩家单位 |
| Marker | 标记 | 用于追踪数值、状态（与 token 常混用） |
| Cube | 方块 | 常见资源计数形态 |
| Track | 轨道 / 条 | 用于记录分数、资源或回合的条状区域 |
| Mat / Player Board | 玩家板 / 个人版图 | 每位玩家自己的整理区 |
| Insert / Organizer | 收纳 | 盒内分隔，与规则无关但玩家常讨论 |

## 5. 资源、货币与标记

| 英文 | 中文 | 释义 |
|------|------|------|
| Resource | 资源 | 游戏中可消耗、转换的抽象或具象物品 |
| Cost | 费用 | 执行行动或打出卡牌所需支付的资源/条件 |
| Income | 收入 | 每轮或触发时获得的资源 |
| Supply | 公共供应堆 | 中央可取用的配件池 |
| Bank | 银行 | 部分游戏对公共金钱或资源的俗称 |
| Limit / Cap | 上限 | 手牌上限、资源上限等 |

## 6. 行动与选择

| 英文 | 中文 | 释义 |
|------|------|------|
| Action | 行动 | 玩家可执行的操作类型 |
| Action Points (AP) | 行动点 | 用行动点预算分配多次小行动 |
| Main Action / Bonus Action | 主行动 / 附加行动 | 主行动消耗大，附加行动常为轻量（依游戏定义） |
| Activation | 激活 | 选择一个单位、卡牌或区域使其生效 |
| Once per turn | 每回合一次 | 频率限制 |
| Optional | 可选 | 非强制 |
| Mandatory | 强制 | 必须执行 |
| Trigger | 触发 | 某条件满足时引发效果 |
| Interrupt | 打断 | 在他人回合插入结算（常见于卡牌异能） |

| 英文 | 中文 | 释义 |
|------|------|------|
| Dice | 骰子 | 随机数工具；**Die** 单数 |
| Roll | 掷 | 投掷骰子 |

| Shuffle | 洗牌 | 随机化牌序 |
| Reveal | 翻开 / 展示 | 由暗转明 |

## 8. 胜利、计分与终局

| 英文 | 中文 | 释义 |
|------|------|------|
| Victory Points (VP) | 胜利分 / 分数 | 常见胜利条件量化 |
| Tiebreaker | 平局决胜 | 同分时比较第二条件 |
| Endgame Trigger | 终局触发 | 满足某条件后进入结算或最后一轮 |
| Majority | 多数控制 | 某区域棋子最多者得分（常见于区域控制） |
| Scoring | 计分 | 即时计分或终局计分 |

## 9. 常见机制类别（中英对照 + 一句话释义）

| 英文 | 中文 | 释义 |
|------|------|------|
| Worker Placement | 工人放置 | 将有限工人放到版图位上，独占或竞争行动位 |
| Deck Building | 牌库构筑 | 游戏过程中不断买牌、删牌，优化自己牌库 |
| Drafting | 轮抽 | 多轮传递牌堆，每人选一留一，兼顾信息与互动 |
| Area Control | 区域控制 | 通过放置单位/影响力争夺区域多数或控制 |
| Engine Building | 引擎构筑 | 建立「产出→再投入」的强化循环，越转越强 |
| Set Collection | 成套收集 | 收集同花色/同数字等组合以得分 |
| Push Your Luck | 搏一搏 / 见好就收 | 可继续冒险获取更高收益或爆掉失去已得 |
| Auction / Bidding | 拍卖 / 竞价 | 用资源或货币竞争购买或顺位 |
| Pick-up and Deliver | 取送 | 从 A 取货送到 B 得分 |
| Route Building | 线路建设 | 修路、铁轨、航线等连通网络 |
| Trick-taking | 吃墩 | 类似桥牌：每墩比较大小，赢墩者得牌权等 |
| Rondel | 轮盘行动轨 | 行动选项排成圆轨，移动距离与代价相关 |
| Roll-and-Write | 掷写 | 掷骰后在纸上勾选或填格 |
| Social Deduction | 社交推理 | 隐藏身份、投票、欺骗与推理 |

## 10. 卡牌与牌库操作

| 英文 | 中文 | 释义 |
|------|------|------|
| Draw | 抽牌 | 从牌库顶取牌 |
| Play | 打出 | 从手牌使用到场上或弃牌堆 |
| Discard | 弃置 | 放入弃牌堆 |
| Exhaust / Tap | 横置 / 耗竭 | 用过一次后旋转或翻面，下回合重置（依游戏） |
| Refresh / Untap | 重置 / 竖回 | 恢复可用状态 |
| Search | 检索 | 从牌库找特定类型牌（常需洗牌） |
| Mill | 磨牌库 | 将牌从牌库顶送入弃牌堆（万智用语，现泛用） |
| Banish / Remove from game | 移出游戏 | 本局不再进入任何牌堆 |

## 11. 版图与移动（战棋 / 美式常见）

| 英文 | 中文 | 释义 |
|------|------|------|
| Hex / Hex Grid | 六边形格 | 六向移动 |
| Square Grid | 方格 | 四向或八向移动 |
| Orthogonal | 正交 | 横竖移动，不含斜向 |
| Diagonal | 斜向 | 对角移动 |
| Range | 射程 / 距离 | 攻击或技能作用格数 |
| Line of Sight (LoS) | 视线 | 中间是否有阻挡 |
| Terrain | 地形 | 不同格子的移动或战斗修正 |
| Miniature | 模型 | 人物或单位造型，常用于战棋 |

## 12. 角色与玩家用语

| 英文 | 中文 | 释义 |
|------|------|------|

## 13. 出版、版本与配件行话

| 英文 | 中文 | 释义 |
|------|------|------|
| Base Game | 基础游戏 | 独立可玩的盒装本体 |
| Expansion | 扩展 | 增加内容，常需基础 |

## 14. 玩家社群常用俚语（非正式，释义用）

| 用语 | 含义 |
|------|------|
| 开荒 | 第一次玩某游戏，边学边玩 |
| 村规 | 非官方、玩家自定的房规 |

## 16. 给 AI 使用的检索提示（元说明）

- 回答**机制类**问题时：优先用本节与第 9 节的定义，再衔接具体游戏规则。  
- 回答**翻译类**问题时：优先采用上表与文中译名，若规则书另有官方译名，以规则书为准。  
- 若用户混用「回合」「轮」「局」，可根据上下文澄清：中文桌游里「一局」常指整盘游戏，「一轮」常指全员各行动一次的大循环（以该款定义为准）。
