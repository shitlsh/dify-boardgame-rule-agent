# 桌游通用术语与概念

跨桌游通用的机制、配件与英文用语说明。单款游戏的专有名词以该款官方规则为准。

# 译名、缩写与歧义

| 主题 | 说明 |
|------|------|
| 中英并用 | 常见写法如「工人放置（Worker Placement）」 |
| 米宝 | 约定译名，对应 Meeple |
| Engine / 引擎 | 在桌游语境多指资源与行动形成的强化循环，非机械引擎 |
| VP | 多为 Victory Points，胜利分 |
| AP | 多为 Action Points，行动点 |
| Round / Turn / 局 | 不同游戏定义不同；中文「一局」多指整盘游戏结束前的全过程 |

# 游戏类型与体验

| 英文 | 中文 | 说明 |
|------|------|------|
| Eurogame / German-style | 德式游戏 | 策略、计分、互动相对克制 |
| Ameritrash / American-style | 美式游戏 | 叙事、模型、冲突与运气成分常更突出 |
| Party Game | 聚会游戏 | 人数多、规则短、偏社交 |
| Abstract | 抽象游戏 | 主题弱、规则极简 |
| Wargame | 战棋 / 兵棋 | 战斗、地形、单位 |
| Living Card Game (LCG) | 成长式卡牌 | 固定扩展、非随机卡包 |
| Trading Card Game (TCG) | 集换式卡牌 | 随机卡包、交易与收藏 |
| Cooperative Game | 合作游戏 | 共同对抗系统 |
| Semi-cooperative | 半合作 | 合作与个人目标并存 |
| Solo / Solitaire | 单人模式 | 单人可玩 |
| 1v1 / FFA / Team | 单挑 / 混战 / 组队 | 对抗结构 |

# 回合、轮次与流程

| 英文 | 中文 | 说明 |
|------|------|------|
| Round | 轮次 | 常指所有玩家各完成一轮行动的大循环；具体以规则定义为准 |
| Turn | 回合 | 某一玩家连续行动的单位；与「轮」易混，以规则为准 |
| Phase | 阶段 | 一轮内按顺序执行的步骤块 |
| Step | 步骤 | 比阶段更细 |
| Initiative / Turn Order | 先手 / 行动顺序 | 先后与方向 |
| Simultaneous | 同时选择 | 同时暗置或同时亮出 |
| Upkeep | 维持 / 回合开始结算 | 自动结算、刷新 |
| Cleanup | 回合结束清理 | 弃牌、重置等 |
| Pass | 跳过 | 宣告不再执行某些行动 |
| Final Round / Last Round | 终局轮 | 结束前的最后一轮或若干轮 |

## 中文常用语对照（易混）

| 说法 | 常见含义 |
|------|----------|
| 一局游戏 | 从开局布置到胜负判定的一次完整对局 |
| 一轮 / Round | 每位玩家各行动一次或完成一轮流程（依规则） |
| 一回合 / Turn | 当前玩家的一次连续行动机会（依规则） |
